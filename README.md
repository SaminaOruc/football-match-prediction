# Football Match Outcome Prediction

This project analyzes English Premier League match data (2000-2025) and applies machine learning to predict match outcomes: home win, draw, or away win.

**Key points:**
- Dataset includes scores, shots, fouls, cards, and other performance metrics.
- Features include cumulative goals, cumulative points, and recent form.
- Models used: Logistic Regression and Random Forest.
- Evaluation: accuracy, precision, recall, F1-score, confusion matrices.
- Best predictions: home wins; draws and away wins are more challenging.
